Featured projects.

How about a few projects we've designed and developed?

Explore Eastern Utah.

We built Explore Eastern Utah to showcase local restaurants that remained open during pandemic shutdowns. We designed the UI in Adobe XD and built it using WordPress and Bootstrap. There is a lot of dynamic content to allow business owners to update their pages. All logos and graphics were designed in Adobe CC.

WordPress, HTML, CSS, JavaScript, Bootstrap, Adobe CC

Arcane Ops. Paranormal Podcast.

Arcane Ops was a limited-run paranormal podcast that ran for seven episodes at the end of 2020. We designed the layout in Adobe XD and built it using WordPress, WooCommerce, and Bootstrap. All logos and graphics were designed in Adobe CC.

WordPress, HTML, CSS, JavaScript, Bootstrap, Adobe CC

Corporate Law Website.

Simple brochure website we designed and built for a corporate law firm. We created the UI in Adobe XD and developed it using Webflow, Finsweet Client First, and custom JS components. Webflow's interface and ease of use are intuitive, but it's a bit restrictive for complex styling. All logos and graphics were designed in Adobe CC.

Webflow, HTML, CSS, JavaScript, Adobe CC

Sheriff Campaign Website.

Campaign website for a sheriff election. This website had many pages, a simple CMS for news updates, and simple PayPal API setup to accept donations. We created the UI in Adobe XD and built it using Bootstrap, HTML, CSS, JavaScript, and some basic PHP to run the CMS. All logos and graphics were designed in Adobe CC.

HTML, CSS, JavaScript, PHP, Bootstrap, Adobe CC
